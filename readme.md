## Code Examples

**Tree sort**:  
Write a function that checks if all letters of a string are unique (case insensitive).

**Solution**

```
const isUnique = s => new Set(s.toLowerCase()).size === s.length;
console.log(isUnique("abcdef"));  // true
console.log(isUnique("Hello"));  // false
```
